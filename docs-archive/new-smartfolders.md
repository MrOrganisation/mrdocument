# New Smart Folders

* There shall be another single configuration file for smart folders in mrdocument root.
* You can specify smart folders by a name, setting the context and (like implemented) conditions and filename regexp.
* You also have to specify a path. If it is relative, it will be relative to mrdocument root. But it can be absolute and be outside of
  mrdocument.
* This will create a single smart folder at the given location.
* The current smart folder configs in the sorted remain. They control the smart folders created in each leaf node.
